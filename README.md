# 🤖 AI-Powered Face Detection & Classification System

An intelligent computer vision application that detects and classifies human faces in real time using Machine Learning and Deep Learning techniques.

---

## 🚀 Overview

This project leverages Artificial Intelligence to automatically detect faces in images and classify them into predefined categories. It combines powerful computer vision techniques with deep learning models to deliver accurate and efficient results.

The system is designed for real-world applications such as security systems, identity recognition, and smart surveillance.

---

## ✨ Features

* 🎯 Real-time face detection
* 🧠 Face classification using trained ML model
* 📷 Image upload and processing
* ⚡ Fast and efficient predictions
* 🌐 Web-based interface

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Computer Vision:** OpenCV
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask

---

## 🧩 How It Works

1. Upload an image through the interface
2. The system detects faces using OpenCV
3. Extracted faces are passed into a trained ML model
4. The model classifies each face
5. Results are displayed with prediction probabilities
